## Dump Media folder

Creates a ZIP archive with media folder content.

```sh
n98-magerun2.phar media:dump [--strip] [<filename>]
```
**Arguments:**
| Argument   | Description   |
|------------|---------------|
| `filename` | Dump filename |
**Options:**
| Option   | Description          |
|----------|----------------------|
| `--strip`| Excludes image cache |
