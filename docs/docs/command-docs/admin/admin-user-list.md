# admin:user:list

List admin users.

```sh
n98-magerun2.phar admin:user:list [--format[="..."]]
```

**Options:**

| Option             | Description                                          |
|--------------------|------------------------------------------------------|
| `--format[=FORMAT]` | Output Format. One of [csv,json,json_array,yaml,xml] |
