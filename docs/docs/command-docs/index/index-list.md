# index:list

Lists all Magento indexes.

## Usage
```sh
n98-magerun2.phar index:list [--format[=FORMAT]]
```

## Options
| Option              | Description                                         |
|---------------------|-----------------------------------------------------|
| `--format[=FORMAT]` | Output Format. One of [csv,json,json_array,yaml,xml] |
