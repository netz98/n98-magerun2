## Toggle template hints

```sh
n98-magerun2.phar dev:template-hints [options] [--] [<store>]
```

**Arguments:**

| Argument | Description      |
|----------|-----------------|
| store    | Store code or ID |

**Options:**

| Option | Description |
|--------|-------------|
| --on   | Switch on   |
| --off  | Switch off  |
