## Toggle allow symlinks setting

```sh
n98-magerun2.phar dev:symlinks [options] [--] [<store>]
```

**Arguments:**

| Argument | Description      |
|----------|-----------------|
| store    | Store code or ID |

**Options:**

| Option     | Description                  |
|------------|------------------------------|
| --on       | Switch on                    |
| --off      | Switch off                   |
| --global   | Set value on default scope   |
