Lists all registered search engines.

```sh
n98-magerun2.phar search:engine:list [--format[=FORMAT]]
```

**Options:**

| Option              | Description                                         |
|---------------------|-----------------------------------------------------|
| `--format[=FORMAT]` | Output Format. One of [csv,json,json_array,yaml,xml] |
