# config:data:di

Print Dependency Injection Config Data.

```sh
n98-magerun2.phar config:data:di [--scope=SCOPE] [<type>]
```

**Arguments:**

| Argument | Description  |
|----------|--------------|
| `type`   | Type (class) |

**Options:**

| Option         | Description                                                                                             |
|----------------|---------------------------------------------------------------------------------------------------------|
| `--scope` `-s` | Config scope (`global`, `adminhtml`, `frontend`, `webapi_rest`, `webapi_soap`, ...) (default: `global`) |
