# config:search

Search in the store config meta data (labels). The output is a table with id, type and name of the config item.

Type can be one of:
- section
- group
- field

```sh
n98-magerun2.phar config:search [--format[="..."]] <search>
```


---

:::note
This command was introduced with version 7.3.0.
:::
