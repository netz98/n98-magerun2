Create a customer token for Webapi.

```sh
n98-magerun2.phar customer:token:create <email> [--no-newline] [<website>]
```

**Options:**

| Option         | Description                        |
|----------------|------------------------------------|
| `--no-newline` | Do not output the trailing newline |
